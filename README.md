# 🚀 Puer Management

A project management and peer evaluation platform designed for university group projects.

Puer Management helps student teams organize projects, manage tasks, track contributions, and conduct fair peer evaluations within a single collaborative environment.

## 📖 Project Overview

Puer Management was developed as part of the CEIT 435 and CEIT 436 Project Development and Management courses at Middle East Technical University (METU).

The platform addresses common challenges in student group projects, including:

- Uneven workload distribution
- Lack of contribution transparency
- Difficulty tracking project progress
- Limited communication of responsibilities
- Fair peer evaluation concerns
